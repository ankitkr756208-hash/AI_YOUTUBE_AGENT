## 📂 Project Structure

```bash
AI_YOUTUBE_AGENT/
│── ui.py
│── youtube_agent.py
│── requirements.txt
│── .gitignore
│── README.md
│── .env
⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/ankitkr756208-hash/AI_YOUTUBE_AGENT.git
cd AI_YOUTUBE_AGENT
2️⃣ Install Dependencies
pip install -r requirements.txt
🔑 Setup Environment Variables

Create .env file:

GROQ_API_KEY=your_groq_api_key_here
▶️ Run App Locally
streamlit run ui.py
🌐 Deploy on Streamlit Cloud
Add Secrets:
GROQ_API_KEY="your_groq_api_key_here"

Then deploy directly from GitHub 🚀

📸 Demo Workflow

1️⃣ Enter YouTube URL
2️⃣ Click Analyze
3️⃣ Get AI Summary + Insights
